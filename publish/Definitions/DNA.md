# Definition
> DNA stands for "deoxyribonucleic acid"
> - Belongs to the family of nucleic acids
# Traits
- Double stranded
	- One "sense" strand
		- Holds the gene
		- 5' to 3'
	- Other "anti-sense" strand
		- Holds the gene's copy
		- 3' to 5'
- Anti-parallel
	- Strands run opposite to each other
